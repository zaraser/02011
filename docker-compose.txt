- ALLER DANS DOCKER COMPOSE ET REGARDER LE SERVICE DOCKER PROXY (volumes)
- taper cette commande dans le terminal --> id -u
- monter le volume " - /run/user/$UID/docker.sock:/var/run/docker.sock:ro "
- remplacer(sans supprimer) le volume correspondant

exemple : - /run/user/101968/docker.sock:/var/run/docker.sock:ro --> morgane
